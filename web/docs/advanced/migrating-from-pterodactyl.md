# Migrating from Pterodactyl
This guide will show you how to migrate from your old Pterodactyl Panel, to your new Calagopus Panel.

## What will be migrated?
Basically everything **EXCEPT** API Keys generated by Pterodactyl, due to Calagopus using a different format than Pterodactyl, both not reversible.

## Installation
This guide has been seperated into 2 different guides depending on how you installed Pterodactyl. Please choose the method that Pterodactyl is installed:
::::tabs
=== Standalone
If you are not using Pterodactyl using Docker or don't know what that is, head to the [Standalone](./migrating/standalone.md) documentation.
=== Dockerized
If you are using Pterodactyl using Docker, head to the [Dockerized](./migrating/dockerized.md) documentation.
::::
