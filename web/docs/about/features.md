# Feature Reference

Calagopus has a *lot* of features. This is why they are split across multiple categories, we will start by listing the most relevant and then we will go into more detail in the following sections.

## Relevant Features

- **Fast**: Calagopus is designed to be fast, the node daemon and the panel backend is written in Rust, the aim is to waste as little resources as possible while still providing a great experience.
- **Secure**: Calagopus is designed to be secure, the architecture has been designed with security in mind, and we have implemented several security features to protect your server and your data.
- **Easy to Use**: Calagopus is designed to be easy to use, the panel is intuitive and user-friendly, and we have implemented several features to make it easy to manage your server and your data.
- **Open Source**: Calagopus is open source, the code is available on GitHub, and we welcome contributions from the community.
- **Cross-Platform**: Calagopus is designed to be cross-platform, the panel can run on essentially any operating system.
- **Extensible**: Calagopus is designed to be extensible, the panel features a plugin system that allows you to extend its functionality and customize it to your needs.
- **Scalable**: Calagopus is designed to be scalable, the architecture allows you to easily scale the panel horizontally and vertically to handle more users and more servers.
- **WebAuthn Authentication**: Calagopus supports WebAuthn authentication, enabling users to log in using biometric data or security keys for a secure and user-friendly login experience.
- **Schedule Tasks**: Calagopus allows you to schedule tasks to run on specific events or at specific times, making it easy to automate server management and maintenance tasks.
- **Subuser Management**: Calagopus supports subuser management, allowing you to create and manage multiple users with different permissions and access levels to your server.
- **File Manager**: Calagopus includes an advanced file manager that allows you to easily manage your server's files, including uploading, downloading, and editing files directly from the panel.
- **Expansive API**: Calagopus provides an expansive API that allows you to interact with the panel programmatically, enabling you to automate tasks and integrate with other tools and services. [See API Reference](https://demo.calagopus.com/api) for more details.

> These features should give you a good overview of what Calagopus has to offer, but there are many more features that we haven't listed here. In the following sections, we will go into more detail about some of the most relevant features and how they can benefit you.
